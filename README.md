# Order Management System
- This is a Order and Stock management system which is manages orders and stocks accurately and orders are not accepts if the stock is not available.
- This application builds with Node.js , Typescript ,Express.js and Mongoose.

##  How to Get Started:
### 1. Clone the repository
```bash
git clone https://github.com/Chaitanya-Rongali/order-managament.git
cd order-management
```

### 2. Install Dependencies 
```bash
npm install
```

### 3. Run the Project 
- Compile the Typescript files
```bash
  tsc
```
- Run the backend server
```
npm start
```
